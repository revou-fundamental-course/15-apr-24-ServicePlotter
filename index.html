How to use?

Sangat mudah. Masukkan nilai, otomatis mengkonversi suhu °C menjadi °F.

Tekan untuk menkonversi sebaliknya. Selamat mencoba!

Formula

Menggunakan: ( S(°C) x 9/5 ) + 32 = S(°F) ( S(°F) - 32 ) x 5/9 = S(°C)
